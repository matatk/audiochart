Interface: DataWrapper

audiochart~DataWrapper

The common interface that the other DataWrappers use. The common DataWrapper 'interface' is validated via the tests. Note: it is not done as a superclass (as PitchMapper is) because there's really nothing in common implementation-wise; only the interface is shared.

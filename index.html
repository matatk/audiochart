Allows the user to explore charts on web pages using sound and the keyboard rather than, or in conjunction with, visually and with the mouse. The code is developed on GitHub and can easily be added to any page containing charts (or HTML tables or JSON data). A tutorial and examples are available.

Thanks to The Paciello Group for allowing me to share this with you, and to The Sonification Handbook for the research behind it.

AudioChart uses the Web Audio API, which has great support amongst browsers, including Microsoft Edge, though is not supported by Internet Explorer.

What does it do for you?

It augments charts you may already have on your page (or those that you create) with sounds that represent the data in the chart.

  • Supports charts made with Google Charts API as well as JSON structures and HTML tables.
  • Can highlight the data visually on the chart during playback (Google Charts only).

More information (including planned work) can be found in the README.

Tutorial and Examples

The Examples Gallery contains line and bar Google Charts, JSON- and HTML table-based examples. Steps to create the Google Charts-based "Hello, world" example are described in the tutorial section of the README.

Code Testing

You can run the unit tests in-browser or via the command line if you have the required tools and package installed (as detailed in the development information section of the README). Also available online are the test coverage details.
